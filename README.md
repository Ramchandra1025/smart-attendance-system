# Smart Attendance System

A web-based Smart Attendance System developed to simplify and digitize attendance management for educational institutions.

## 🚀 Features
- Secure Student & Faculty Login
- OTP-based Registration & Verification (EmailJS)
- Real-time Attendance Tracking
- Attendance Percentage Calculation
- Attendance Shortage Detection (<75%)
- Student Request System (Leave/Explanation with Approval)

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- Supabase (Backend & Database)
- EmailJS (Email OTP Service)
